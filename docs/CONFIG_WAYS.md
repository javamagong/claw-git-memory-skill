## 📋 配置场景

### 场景 A：单设备使用

```
配置：
- 远程同步：❌ 不启用
- 多设备同步：❌ 不启用

使用：
- 本地自动保存
- 历史版本可查
- 零配置
```

---

### 场景 B：云端 + 本地双设备

```
云端配置：
- 远程同步：✅ 启用
- 多设备同步：✅ 启用
- 设备类型：cloud
- 冲突解决：远程优先

本地配置：
- 远程同步：✅ 启用
- 多设备同步：✅ 启用
- 设备类型：local
- 冲突解决：本地优先
```

---

### 场景 C：远程备份

```
配置：
- 远程同步：✅ 启用
- 多设备同步：❌ 不启用
- 自动推送：✅ 启用

使用：
- 自动备份到 GitHub
- 本地优先
- 不自动拉取远程
```
